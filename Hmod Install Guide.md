## Hmod
Hmod can be installed by [patching the server jar](https://github.com/Username7495/Mods-and-Tweaks-for-Cypress/blob/main/Jar%20Patching.md) or can be run via console.

Example: "java -classpath hmod_QoL.jar;ext1605_20_server_QoL.jar -Xmx1024M -Xms1024M Main".
