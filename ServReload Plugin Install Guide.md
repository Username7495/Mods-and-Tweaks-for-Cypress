## ServReload Plugin
A plugin for Hmod that automatically reloads the server every hour. In order to work the plugin requires [ServReload Mod](https://github.com/Username7495/Mods-and-Tweaks-for-Cypress/blob/main/ServReload%20Mod%20Install%20Guide.md).


  
### How to use
  ServReload is [used like any other Hmod plugin](https://github.com/Username7495/Mods-and-Tweaks-for-Cypress/blob/main/Using%20Plugins.md), you can configure reload time via its .properties file.
